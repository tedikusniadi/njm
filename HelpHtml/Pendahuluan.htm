ERP System Overview

 

Bhumidana ERP system merupakan aplikasi berbasis web yang meliputi modul – modul financial akunting, penjualan, pembelian, inventory dan human resource yang saling berkesinambungan untuk membantu perusahaan menjalankan operasi bisnisnya secara otomatis dan terintegrasi.

 

Petunjuk penggunaan ini diharapkan dapat membantu para pengguna aplikasi untuk dapat mengoperasikan sistem ERP ini dengan baik.

 

Para pengguna disarankan untuk menggunakan Internet browser Firefox Mozilla versi 1.0 keatas untuk mengakses sistem ERP ini. Untuk mengakses egati ERP, masukan URL http://202.138.238.9/AccountingService.
