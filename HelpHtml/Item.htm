Item

Kode Menu : I12

Navigasi : Inventory > Item Management > Item

 

Fungsi: untuk mengelola varian barang barang

Data

Keterangan

Item ID

Kode unik barang

Item Name

Nama barang

Item Category

Kategori barang. Daftar pilihan diambil dari menu: Inventory > Item Management > Item Category

Currency

Satuan mata uang

Standard Price

Harga standar barang

Description

Deskripsi barang

Status

Active, jika barang aktif diperjual belikan

Inactive, jika barang sudah tidak diperjual belikan

 

 

Tab attribute, menyimpan informasi mengenai atribut barang

Data

Keterangan

Base UoM

Satuan ukur dasar.

Unit Weight (Kg)

Berat dalam satuan kilogram. Data ini hanya dapat dimasukkan jika Base UoM yang dipilih bukan merupakan Base Measurement (referensi ke menu : Company Administration > General Setting > Unit of Measure)

Unit Width (M)

Lebar dalam satuan meter. Data ini hanya dapat dimasukkan jika Base UoM yang dipilih bukan merupakan Base Measurement (referensi ke menu : Company Administration > General Setting > Unit of Measure)

Unit Length (M)

Panjang dalam satuan meter. Data ini hanya dapat dimasukkan jika Base UoM yang dipilih bukan merupakan Base Measurement (referensi ke menu : Company Administration > General Setting > Unit of Measure)

Unit Height (M)

Tinggi dalam satuan meter. Data ini hanya dapat dimasukkan jika Base UoM yang dipilih bukan merupakan Base Measurement (referensi ke menu : Company Administration > General Setting > Unit of Measure)
