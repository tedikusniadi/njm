Picking Order

Kode Menu : S52

Navigasi : Sales > Delivery Management > Picking Order

 

Fungsi: untuk mengelola pengambilan barang yang akan dikirim untuk memenuhi pesanan pelanggan. Picking order dibuat berdasarkan rencana pengiriman. Untuk memudahkan pengguna aplikasi, ketika akan dibuat picking order akan ditampilkan daftar rencana pengiriman yang telah disusun berdasarkan tanggal rencana pengiriman.

Data

Keterangan

Picking Order ID

Kode unik pengambilan barang

Company Facility Code

Kode fasilitas bisnis yang mengelola pengambilan barang

Date

Tanggal pengambilan barang

Customer

Nama pelanggan

Delivery planning No

Nomor rencana pengiriman. Daftar pilihan diambil dari data delivery plan milik pelanggan yang telah dipilih sebelumnya dan delivery plan tersebut belum pernah dibuat picking order

Claim / Return Status

Yes, jika terjadi klaim / return atas kiriman ini.

No, jika tidak terjadi klaim / return atas kiriman ini.

Data ini terisi otomatis oleh sistem

 

Tab picking order item, dapat diisikan data barang yang diambil untuk dikirim

Data

Keterangan

Item

Nama barang. Daftar pilihan diambil dari data delivery plan yang telah dibuat sebelumnya

Picked Unit

Banyaknya barang yang diambil

UoM

Satuan unit barang. Data ini terisi otomatis

Total (kg)

Banyaknya barang yang diambil dalam satuan kilogram. Data ini terisi otomatis

Description

Keterangan tambahan
