User

Kode Menu : T12

Navigasi : Tools > User Management > User

 

Fungsi: untuk mengelola akun pengguna sistem.

Data

Keterangan

User ID

Nomor akun pengguna sistem. Nomor ini merupakan nomor pegawai

Password

Password pengguna sistem

Person

Pegawai yang datanya telah disimpan pada menu : Human Resource > Personnel > Personnel Management

Note

Keterangan tambahan

Status

Active, jika akun pengguna sistem masih berlaku

Inactive, jika akun pengguna sistem tidak berlaku atau dibekukan

Role

Pada bagian ini dipilih peran untuk akun pengguna sistem ini. Daftar pilihan diambil dari menu: Tools > User Management > Role
