Goods Receipt

Kode Menu : I51

Navigasi : Inventory > Inventory transaction > Goods Receipt

 

Fungsi: untuk mengelola pencatatan barang masuk ke gudang

Data

Keterangan

Goods Receipt ID

Kode unik pencatatan barang masuk

Company Facility Coder

Kode fasilitas bisnis yang menerima barang

Receipt Date

Tanggal penerimaan barang

Reference Type

Surat referensi penerimaan barang. Referensi penerimaan barang dapat berasal dari: Purchase Order, Transfer Order atau Sales Return

Reference No

Nomor surat referensi

Note

Keterangan tambahan

 

Setelah memilih surat referensi, berikutnya dicatat tempat penyimpanan barang dalam gudang

Data

Keterangan

Rejected Unit

Banyaknya barang yang ditolak

Container ID

Nama container tempat penyimpanan barang

Quantity

Banyaknya barang yang ada pada container

Received Unit

Banyaknya barang yang disimpan dalam kontainer
