Sales Return

Kode Menu : S62

Navigasi : Sales > Sales Return Management > Sales Return

 

Fungsi: untuk mencatat pengembalian barang.

Keterangan

Sales Return Action

Tindakan terhadap pengembalian barang.

Update delivery planning, jika akan dikirimkan barang baru untuk mengganti barang yang telah dikembalikan

Update sales order, jika barang yang telah dikembalikan tidak akan digantikan. Update sales order akan

Delivery Order No

Nomor delivery order daripada barang yang dikembalikan
