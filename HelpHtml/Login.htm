Login

 

Halaman login akan muncul pertama kali ketika mengakses sistem ERP ini. Fitur user management mengatur pengguna serta hak aksesnya ke dalam sistem ERP ini.

 

Gambar1 Halaman Login

 

Pengguna aplikasi harus terlebih dahulu didaftarkan ke dalam fitur user management oleh sistem administrator. Untuk dapat lebih lanjut mengakses sistem ERP, pengguna harus memasukkan user ID dan password.
