Delivery Order

Kode Menu : S53

Navigasi : Sales > Delivery Management > Delivery Order

 

Fungsi: untuk mengelola perintah pengiriman barang. Perintah pengiriman barang dibuat berdasarkan picking order. Untuk memudahkan pengguna aplikasi, ketika akan dibuat delivery order akan ditampilkan daftar picking order yang telah selesai dibuat

Data

Keterangan

Delivery Order ID

Kode unik perintah pengiriman barang

Company Facility Code

Kode fasilitas bisnis yang mengelola perintah pengiriman barang

Date

Tanggal pembuatan rencana

Picking Order ID

Kode picking order

Customer

Nama pelanggan

Total Weight (kg)

Berat total dalam satuan kilogram

 

Tab customer, berisi tentang data pelanggan

Data

Keterangan

Shipping Address

Alamat pengiriman barang, terisi otomatis berdasarkan data yang dimasukkan pada delivery plan

Postal Code

Kode Pos, terisi otomatis berdasarkan pilihan shipping address

City

Kota, terisi otomatis berdasarkan pilihan shipping address

Country

Negara, terisi otomatis berdasarkan pilihan shipping address

 

Tab delivery order item, berisi tentang data barang yang dikirim

Data

Keterangan

Delivery Order Item

Nama barang

Delivered Unit

Banyaknya barang yang dikirim

UoM

Satuan unit barang

Weight (kg)

Banyaknya barang yang dikirim dalam satuan kilogram. Data ini terisi otomatis
